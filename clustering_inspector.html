Clustering Inspector¶

Convenience class used to review the results gathered from weighted and unweighted clustering routines.

Together with reviewing the results of clustering experiments, it is possible to perform pathway enrichment (through TargetMine’s API) for the cluster containing a user specified gene. The enrichment results are only displayed to screen.

init_service(service=None, user=None, pwd=None, token=None)[source]¶

Inialize the Service used for results enrichment

In order to perform enrichment of the clusters found within the network, we need to connect to TargetMine’s enrichment services. These are provided, for registered users, through a public API. This method uses the provided user and password credentials to create a service that can later be used to perform cluster enrichment. Currently, two services are available, both based on the InterMine platform: TargetMine (for Human PPI clustering results) and FlyMine (for Drosophila PPI clustering results). It also defines the default options used to perform cluster enrichment through the TargetMine service.

Parameters

  • service (str) – The enrichment service to use

  • user (str) – The user name registered in the service

  • pwd (str) – The password for the given user

  • token (str) – Alternative to user/pwd login, the token registered to use a given account in the corresponding service
